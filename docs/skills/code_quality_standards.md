# Skill: Code Quality

O código deve ser escrito em nível profissional.

## Regras obrigatórias
- Nomes claros e sem abreviações confusas
- Funções pequenas e com responsabilidade única
- Nenhum código morto ou TODO esquecido
- Sem duplicação de lógica

## Organização
- Widgets grandes devem ser quebrados
- Componentes reutilizáveis devem ser extraídos
- Comentários apenas quando agregam contexto real

## Manutenção
- Código deve ser fácil de entender sem explicação externa
- Mudanças não devem quebrar fluxos existentes
